## ✨ Overview

This project builds a **complete Hindi Automatic Speech Recognition (ASR) system** using Whisper.

💡 It goes beyond standard pipelines by introducing:
- Fine-tuning for domain adaptation  
- Smart post-processing pipeline  
- Word-level error intelligence  
- Lattice-based evaluation (advanced concept)  

---

## 🧩 Problem

Traditional ASR systems:
- ❌ Penalize valid variations  
- ❌ Ignore linguistic flexibility  
- ❌ Fail in multilingual setups  

---

## 🚀 Solution

✔ Fine-tuned Whisper for Hindi  
✔ Built intelligent cleanup pipeline  
✔ Designed word classification system  
✔ Implemented lattice-based evaluation
